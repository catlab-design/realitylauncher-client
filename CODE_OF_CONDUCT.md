# Code of Conduct

## Our Pledge

We as contributors and maintainers pledge to make participation in this project a harassment-free experience for everyone.

## Our Standards

Examples of behavior that contributes to a positive environment:
- Being respectful and constructive.
- Accepting and giving feedback professionally.
- Focusing on what is best for the community and project.

Examples of unacceptable behavior:
- Harassment, discrimination, or personal attacks.
- Trolling, insulting, or derogatory comments.
- Publishing private information without explicit permission.
- Any conduct that is inappropriate in a professional setting.

## Enforcement Responsibilities

Project maintainers are responsible for clarifying and enforcing this Code of Conduct and may remove or edit comments, commits, code, issues, and other contributions that violate it.

## Scope

This Code of Conduct applies within project spaces and in public spaces when an individual is officially representing the project.
