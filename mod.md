Fresh Animations is a work in progress resource pack that gives an animation overhaul to the  creatures in the default look of Minecraft. The aim is to make the mobs more dynamic and believable.    


---
  
![Optifine or EMF Required](https://imgur.com/h2JOFrT.png)

This resource pack uses an OptiFine feature called custom entity models (CEM) to add new models and animations.  
**IT WILL NOT WORK PROPERLY WITHOUT THIS FEATURE!**  
As an alternative to Optifine, the combination of _Entity Model_ Features and _Entity Texture Features_ can be used for CEM features.

<details>
  <summary>Frequently Asked Questions:</summary>  
  
### Is Fresh Animations a mod or datapack?
> Nope, Fresh Animations is just a resource pack! It does require OptiFine features to work, though.

### Is there a way to use Fresh Animations without OptiFine?
> Fresh Animations requires OptiFine's Custom Entity Models (CEM) and CEM Animation features to work. 
> As an alternative to these features, I highly recommend *[Entity Model Features (EMF)](https://modrinth.com/mod/entity-model-features)* and *[Entity Texture Features (ETF)](https://modrinth.com/mod/entitytexturefeatures)*.

### When will the next update come out?
> Whenever I feel happy with the new content. Please don't ask for a release date. 
> For those willing to pay, Golem Tier Ko-fi supporters get updates early.

### Why do some mobs have blank eyes and no new animations when I load the resource pack?
> This means either that the required mods aren't being used, or there is a conflict with another mod.
> - When using OptiFine make sure you install it correctly, and that Custom Entity Models is enabled. 
> The required OptiFine settings are: *Video Settings > Quality > Custom Entity Models: ON*
> - When using EMF & ETF, it's likely a mod or resource pack conflict. First confirm it is by using only EMF, ETF and FA. 
> If the issue persists, reset your EMF settings. If it fixes it, try find the conflicting mod. 
> Otherwise you can ask for help on my Discord server.
> - This can also happen with some 3rd party clients.

### Is Fresh Animations compatible with *this other resource pack*?
> Probably not completely. Resource packs that only edit textures will probably be fine, but packs with entity models won't work as it would override FA's models and animations.
> If a pack doesn't work, there may be an FA patch version available so check the pack's description or look it up on CurseForge or Modrinth.

### Is Fresh Animations compatible with *this mod*?
> Some mods have issues with FA. There is a list of these mods on my Discord server, as well as a help channel for reporting and sorting through mod issues.
> If you find an issue with a mod, consider reporting it via my Discord server.

### Can you add player animations?
> I have started developing a Player Add-on for Fresh Animations, which is currently available on my [Ko-fi](https://ko-fi.com/freshlx) for Golem Tier members. It requires EMF & ETF as OptiFine doesn't support custom players.

### Why isn't *this mob* animated? Can you add it?
> Not all the mobs are animated yet. New mobs animations will be added in major updates.
> I work and add new animations at my own pace, so please refrain from requesting animations.

### Can you make a Bedrock port?
> It's not as simple as porting it. To make a version that would work on Bedrock, I would have to start from scratch and it would end up being completely different. So that's not something I plan to do anytime soon.

### Why do some mobs have purple/black textures when I load the resource pack?
> This is an OptiFine bug that can occur when the pack is loaded in. To solve this, you need to exit the game and re-launch it.

### The pack is warning me that it's made for another version of Minecraft. Which version should I get?
> Any version of this pack should work mostly as intended in most game versions it's tagged with on CurseForge or Modrinth, as long as there's a version of OptiFine or EMF to run it. Don't worry about the version warning.
</details>
